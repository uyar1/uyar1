## What I've built

**[Parliamentary Protocol AI](https://github.com/uyar1/parliamentary-protocol-ai)** — A multi-service system that automatically transcribes and summarizes parliamentary sessions for the Bremen Parliament. FastAPI backend orchestrating WhisperX (speech-to-text), quantized LLMs (Llama 3.1, DeepSeek), and a multi-stage summarization pipeline with tournament-style evaluation. Dockerized with NVIDIA GPU acceleration.

**[Document Q&A (RAG)](https://github.com/uyar1/document-qa-rag)** — Upload PDFs and ask questions about their content. RAG pipeline with FastAPI, ChromaDB, OpenAI Embeddings, and Claude for answer generation. Deployed on AWS.

**EvoAl-LLM** *(thesis, code not yet public)* — Fine-tuned Qwen3-4B with LoRA/QLoRA to generate code in a domain-specific language for evolutionary algorithms. Synthetic dataset (7,500 examples), Flask REST API with OpenAI-compatible interface, VS Code integration.
